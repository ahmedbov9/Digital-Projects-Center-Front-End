# Digital Project Center

A digital platform specialized in providing web development services, graduation project implementation, technical consultations, and customer support.  
A professional interface built with **React** and **MUI** featuring a user management system and a comprehensive admin dashboard.

---

## Main Features

- **Web Development**: Build modern, responsive websites using the latest technologies.
- **Graduation Projects**: Implement high-quality, professional graduation projects.
- **Technical Consultations**: Technical support and consultations for projects and individuals.
- **Service Request System**: Integrated service request form with file upload and order tracking.
- **Admin Dashboard**: Manage orders and users with advanced statistics.
- **Authentication System**: Login/Register/Change Password/Reset Password.
- **Interactive Alerts**: Using SweetAlert2.
- **Arabic Language Support**: Fully Arabic interface with professional Arabic fonts.
- **Responsive Design**: Compatible with all devices and screens.

---

## Technologies Used

- [React](https://reactjs.org/)
- [Material UI (MUI)](https://mui.com/)
- [React Router](https://reactrouter.com/)
- [Axios](https://axios-http.com/)
- [SweetAlert2](https://sweetalert2.github.io/)
- [FontAwesome](https://fontawesome.com/)
- [Bootstrap](https://getbootstrap.com/) (for some elements)
- Arabic fonts (Almarai, Noto Sans Arabic, etc.)

## Getting Started

1. **Install dependencies:**

   ```bash
   npm install
   ```

2. **Run the app locally:**

   ```bash
   npm start
   ```

   Then open [http://localhost:3000](http://localhost:3000)

3. **Environment setup:**
   - Edit the `.env` file according to your API configuration.

## Important Files

- **robots.txt**: Prevents indexing of private pages (dashboard, orders, etc.).
- **sitemap.xml**: Specifies public pages for search engines.
- **.env**: Environment settings (do not share this file publicly).
